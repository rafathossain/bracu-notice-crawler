### Key Features

#### Data Extraction
BRACU crawler can extract the following data while crawling:

- Notice Headline
- URLs
- Publish Date
- Notice Description

This projects fetches the data from BRAC University website and shows it in a table.

Brac University publishes it's regular notices in the following URL.

Notice URL: https://www.bracu.ac.bd/news-events/news-archive?field_news_department_tid_selective=46

<p align="center">
    <img src="http://www.phpzone.in/wp-content/uploads/2017/08/php-curl-request-665x251.png" alt="demo" width="60%" border="0">

This php and cUrl based crawler gets all the data from the URL and shows them in a table.
